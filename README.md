# lx-vue-element-admin

## command
npm 命令 | 描述
:-:|:-:
``npm install`` | 安装相关依赖
``npm run serve`` | 启动开发环境
``npm run build`` | 构建打包代码
``npm run build:test`` | 构建打包代码至测试服务器
``npm run icon`` | 快速生成开发字体图标css
``npm run build:icon`` | 快速生成正式字体图标css及字体文件
